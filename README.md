## 🛠️ Tech Stack & Coding Philosophy

The application is built with performance, transparency, and clean code optimization at its core:

* **Core UI Architecture:** React 18+ (Integrated via CDN utilizing pure `React.createElement` architecture, ensuring deep understanding of the reconciliation engine, zero reliance on opaque compilation layers, and a lightweight footprint).
* **Styling & Design System:** Tailwind CSS (Leveraging utility-first utility classes to maintain strict mobile-first visual hierarchies, crisp modern typography appropriate for professional medical software, and fluid responsive states).
* **Foundations:** Semantic HTML5 (crucial for healthcare accessibility standards) and modern asynchronous JavaScript (ES6+).
* **Version Control:** Git & GitHub workflow pipelines.
